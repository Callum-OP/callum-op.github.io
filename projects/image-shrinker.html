Image Shrinker

August 2025 - Personal Project

GitHub Repo: https://github.com/Callum-OP/Image-Shrinker
View GitHub Pages website: https://callum-op.github.io/Image-Shrinker



JavaScript, HTML, CSS

A javascript website for shrinking one or more images. User can choose by how much percent they want to decrease the size of the image. Useful if you have several large images and need to decrease their size significantly.

I made this as I needed to shrink several images that I intended to use as game assets but they were far too large.

Features:
  • Shrink an image from a percent of your choice.
  • Shrink several images at once.
  • Download images, if there are over 5 images they are put in a zip folder.
