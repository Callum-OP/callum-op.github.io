Batch Image Editor

Jan 2026 - Personal Project

GitHub Repo: https://github.com/Callum-OP/Batch-Image-Editor



An app that allows the batch processing of several images at once.

Features:
  • Resize images, with option to keep the ratio or not.
  • Reformat images, changing them from jpg to png, png to jpg, etc.
  • Can choose to edit images individually or all at once.
