Spritesheet Reader

July 2025 - Personal Project

GitHub Repo: https://github.com/Callum-OP/Spritesheet-Reader
View GitHub Pages website: https://callum-op.github.io/Spritesheet-Reader



JavaScript, HTML, CSS

A simple app that takes a json data file for a spritesheet and then displays the coordinates for each texture, for animations named (as an example) run1-run6 then it will display the start and end coordinates for those textures in the spritesheet. I made this to make it easier to get the values for animations within a spritehseet so that I could easily iterate through them from start to finish. The output is ready to be copied and pasted into code.

I decided to host it online to make it easier for me to access so that I don't need to find the code and the run the program each time, also made the app adaptable and able to take in json data from several sources. Look below to see examples of animations such as jogging and sheathing and how the app gets the coordinates for the first texture and then gets the coordinates for the last texture of each direction.
