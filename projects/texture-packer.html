Texture Packer

Aug 2025 - Personal Project

GitHub Repo: https://github.com/Callum-OP/Texture-Packer
View GitHub Pages website: https://callum-op.github.io/Texture-Packer



JavaScript, HTML, CSS

A javascript website for making spritesheets, allows you to select multiple files and then download the completed spritesheet and JSON data file.

I made this to easily create spritesheets for game development and to make spritesheets compatible with my spritesheet reader app so that I can make an animations easily.

Features:
  • Pack several images into a single spritesheet or atlas.
  • Download the spritesheet image and the JSON data file that points to where each texture is.
