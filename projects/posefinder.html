React, Vite, JavaScript, HTML, CSS, Tailwind CSS

Built a browser app that will extract the pose data from a video or image using MediaPipe ML models and allow it to be exported as a BVH file. The file can then be used in 3D software to animate a character.

Features:
  • Upload videos or images.
  • Frame inspector and option to select which person to track.
  • Export pose data to BVH or JSON.
  • Settings panel to customise how the app collects pose data.
