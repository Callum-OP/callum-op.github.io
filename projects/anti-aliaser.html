Anti Aliaser

July 2025 - Personal Project

GitHub Repo: https://github.com/Callum-OP/Anti-Aliaser
View GitHub Pages website: https://callum-op.github.io/Anti-Aliaser



JavaScript, HTML, CSS

An anti aliasing tool that takes an image of a line drawing and then thins the lines, and fades the edges to create an effect similar to anti-aliasing. I made this as I wanted to apply anti aliasing to old line drawings without needing to redraw them. I made it online using Javascript to make it easier to access.

I had already made an app like in Python but it was annoying trying to manually find the file and run the code, by putting it on the web all I need is to paste the image and it runs instantly.

Features:
  • Use file explorer to select a file.
  • Paste an image directly from the clipboard.
  • Download the processed image.
  • Copy the the processed image to the clipboard.
  • View processed image in browser before saving.
