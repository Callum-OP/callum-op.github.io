3D Object Generator

Jan 2026 - Feb 2026 - Personal Project

GitHub Repo: https://github.com/Callum-OP/Gemini-AI-3D-Generator
View GitHub Pages website: https://callum-op.github.io/Gemini-AI-3D-Generator



JavaScript, HTML, CSS, Bootstrap

A javascript website that takes a prompt and API key from user, then sends the prompt to Gemini 3 Flash with detailed instructions to create Three.js code which creates something close to what the prompt described.

Allows anyone to quickly get a downloadable low poly 3D object that they can use and edit in Blender.

Features:
  • Generate a 3D object and inspect it in the 3D viewer.
  • Download the 3D generate object and use it in other tools.
