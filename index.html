My thoughts on AI

Only recently after graduating university have I experimented with using AI tools such as Microsoft Copilot, often for debugging existing code and seeing how a feature could be implemented. When it comes to making applications from scratch, AI is useful especially for smaller, single file applications, allowing me to get a quick base for app and then modify it to work how I want it to.

However when it comes to making larger applications or editing existing applications AI often became unreliable and would forget the context of the app. It also depends on the programming language being used, I have noticed Copilot was quite good at Python and also able to generate reasonable JavaScript and HTML/CSS but when it came to languages such as TypeScript and C++ the code it generated was often not reliable and would only slow me down.

I have a similar opinion on AI for art, where it is good at generating quick concept images but usually not good enough to be the final product.
